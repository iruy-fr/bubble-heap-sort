# Changelog

## [1.1.0] 
### Adicionado
- Implementação de testes com vetores ordenados
- Implementação de testes com vetores inversamente ordenados
- Estrutura para múltiplos tipos de entrada (aleatório, ordenado, inverso)
- Geração de gráficos separados por tipo de vetor

### Melhorado
- Organização dos testes experimentais
- Clareza na separação dos cenários de análise

### Observações
- Agora o projeto atende aos requisitos completos do trabalho acadêmico

## [1.0.0] - 
### Adicionado
- Implementação do Bubble Sort (O(n²))
- Implementação do Heap Sort (O(n log n))
- Geração de vetores aleatórios
- Medição de tempo de execução
- Execução de múltiplos testes
- Cálculo da média dos tempos
- Geração de gráfico comparativo

### Resultados
- Heap Sort apresentou desempenho significativamente superior ao Bubble Sort
- Diferença cresce conforme o tamanho do vetor aumenta

### Observações
- Testes realizados inicialmente apenas com vetores aleatórios
